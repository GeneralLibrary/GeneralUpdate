# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in GeneralUpdate, please report it by emailing the project lead at **zhuzhen723723@outlook.com**.

Please **do not** create a public GitHub issue for security vulnerabilities.

We will acknowledge receipt of your report within 72 hours and provide an estimated timeline for a fix and release.

## Disclosure Policy

- We will investigate and validate the report within 7 business days.
- A fix will be prepared and tested before any public disclosure.
- A security advisory will be published on GitHub once the fix is released.
- We will credit the reporter if they wish to be acknowledged.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| Latest major release | ✅ Yes |
| Older versions | ⚠️ Limited (critical fixes only) |
