# GeneralUpdate #

**更新无限，升级无界。**</br>

GeneralUpdate是一款基于.NET Standard2.0 Apache 2.0协议开源的跨平台应用程序自动升级组件。</br>

不依赖任何UI框架，减少更新产生的资源消耗。</br> 

一键启动示例帮助您快速理解如何为您的应用程序集成自动升级能力。
